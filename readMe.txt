------------------------------------------------------------------------------------------------------------------------------------------------
This project is part of a thesis with title "Μελέτη και Υλοποίηση ενός Συστήματος Αξιολόγησης και Λογοδοσίας Μοντέλων Αυτόματης Λήψης Αποφάσεων"
------------------------------------------------------------------------------------------------------------------------------------------------
The purpose of the project is to evaluate the robustness of models using adversarial attacks and defenses via the ART Library
------------------------------------------------------------------------------------------------------------------------------------------------
Current implementation only supports models of ResNet-18 architecture but users can easily modify the code to include more
------------------------------------------------------------------------------------------------------------------------------------------------
The dataset of this project is PatchCamelyon (PCam) which can be found here->
https://drive.google.com/drive/folders/1gHou49cA1s5vua2V5L98Lt8TiWA3FrKB?usp=drive_link
------------------------------------------------------------------------------------------------------------------------------------------------
To run the system run the Main.py class and then run this command in the terminal->
streamlit run [...]/Main.py
